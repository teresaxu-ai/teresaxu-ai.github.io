Project Overview

The AI Investment Chatbot is a personal project exploring how conversational AI can make financial analysis more accessible. It handles investment queries, market context, and trading-style insights using an LLM-backed pipeline — demonstrating practical application of modern language models in a financial context.

The Challenge

Much of a risk or investment analyst’s time can go into pulling and normalizing real-time data instead of actually analyzing it. The spark for this project was to explore whether a single conversational interface could shift that balance — letting users ask questions in plain language and get back structured, data-backed insights.

Traditional tools also assume a lot of domain knowledge. Retail investors often struggle with:

  • Complex financial data interfaces that require domain knowledge
  • Time-consuming research processes across multiple platforms
  • Difficulty translating market data into actionable insights
  • Lack of personalized guidance for investment decisions

The goal was to create an intelligent interface that could democratize access to financial analysis through natural conversation.

System Design

The architecture follows a microservices pattern with clear separation of concerns:

  • API Gateway: FastAPI handles all incoming requests and routing
  • LLM Integration: Sophisticated prompt engineering for financial context
  • Data Layer: Real-time financial data aggregation and processing
  • Response Engine: Intelligent formatting of complex financial data

Key Features

Natural Language Processing

Advanced NLP capabilities allow users to ask complex questions in plain English, such as "What are the risks of investing in tech stocks during market volatility?" and receive comprehensive, tailored responses.

Interactive Data Exploration

Generated code is fully visible and editable, allowing users to modify or extend the analysis. The platform includes a smart data pipeline with cache-first retrieval and fallback to Yahoo Finance via yfinance for real-time data, supports CSV uploads for custom datasets, and renders charts instantly in-browser using Matplotlib.

Risk Assessment

Sophisticated risk modeling provides users with clear understanding of potential investment risks, presented in accessible language rather than complex financial jargon.

Results & Impact

The chatbot demonstrates how conversational AI can make financial analysis more accessible: users ask questions in plain language and receive structured insights without switching between multiple tools. As a personal project, the focus was on validating the end-to-end flow — from natural language input through LLM processing to actionable output — rather than formal metrics.

Key Learnings

LLM Optimization

Discovered that financial domain-specific prompt engineering significantly improves response accuracy. Generic prompts often resulted in overly cautious or generic advice.

User Experience Design

Learned the importance of managing user expectations in AI interactions. Clear communication about the chatbot's capabilities and limitations increased user trust and satisfaction.

Data Quality Impact

Real-time data quality directly affects user trust. Implementing robust data validation and error handling became crucial for maintaining credibility in financial recommendations.
